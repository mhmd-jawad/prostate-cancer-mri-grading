\section*{Introduction}
This report presents the evaluation of the BAMF/AIMI prostate MR pretrained segmentation pipeline on the PROMISE12 test set. PROMISE12 contains T2-weighted prostate MRI scans with binary whole-gland segmentation masks. In this experiment, the BAMF/AIMI model was used to generate prostate segmentation masks for the PROMISE12 test cases, and the predictions were compared against the corresponding ground-truth masks. The main evaluation metric is the Dice Similarity Coefficient (Dice), computed for each case and summarized across the dataset.

\section*{Summary}
The BAMF/AIMI pipeline achieved a mean Dice score of 0.9163 on the evaluated PROMISE12 test cases. Across 30 evaluated cases, the model showed consistently strong whole-gland prostate segmentation performance, with some variation between easier and more challenging scans.
